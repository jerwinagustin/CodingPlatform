About This Project

This is a Django starter project, built similar to create-vite but for Django.

Features

  • Clean project structure
  • Pre-configured settings
  • Sample Post model
  • Admin panel ready
  • Basic templates and static files

Tech Stack

  • Django 5.x
  • SQLite (easily swappable to PostgreSQL/MySQL)
  • Python 3.8+

Built with Django
